# Budget Lens - アプリケーション実装状況レポート

このドキュメントは、予算・支出管理アプリケーション「**Budget Lens**」の現在の技術スタック、ディレクトリ構造、機能実装状況を詳細にまとめたものです。Notebookなどの共有・記録用ドキュメントへそのままコピー＆ペーストしてご利用いただけます。

---

## 1. プロジェクト概要 & 技術スタック

「Budget Lens」は、リアルタイムで収支を追跡し、スマートな予算管理をサポートするWebアプリケーションです。

*   **フロントエンドフレームワーク**: Next.js v16.2.9 (App Router 構成)
*   **ライブラリ**: React v19.2.4
*   **パッケージマネージャー**: `pnpm` (v10.15.1)
*   **データベース / 認証**: Supabase (`@supabase/ssr` を採用した認証・データ操作)
*   **スタイリング**: Tailwind CSS v4.3.1
    *   *※`tailwind.config.js` は使用せず、`src/app/globals.css` の `@theme inline` でカスタムテーマを直接定義しています。*
*   **コードフォーマッター / Linter**: Biome

## 3. 機能別の実装ステータス

### 🔑 3.1. ユーザー認証・アクセス制限 (Authentication)
*   **ログイン / 新規登録画面 (`/login`)**:
    *   メールアドレスとパスワードによる認証UIを提供。
    *   `src/app/login/actions.ts` で Supabase の認証APIを呼び出す Server Actions（ログイン・ログアウト）を実装。
*   **ルート保護 (Middleware)**:
    *   `src/lib/supabase/middleware.ts` により、未ログインユーザーが `/dashboard` などの保護されたルートにアクセスした際に自動で `/login` へリダイレクトする仕組みを実装。
*   **ユーザー情報取得フック (`use-current-user.ts`)**:
    *   クライアントサイドで現在ログインしているユーザー情報（ID、Email等）を安全に参照・監視するためのカスタム React フック。

### 📊 3.2. ダッシュボード (Dashboard)
*   **クイック統計カード**:
    *   「今月の残り予算（全体予算に対する未消化割合）」、「今月の総支出（前月比%の算出付き）」、「今月の手取り収入」を表示。
    *   計算結果はプログレスバーで視覚的にも進捗率を表示。
*   **直近の支出履歴**:
    *   ユーザーが登録した最新 of 支出レコードを直近4件までダッシュボード上にリスト表示。
*   **カテゴリ別支出割合（円グラフ）**:
    *   当月の支出データをカテゴリIDでグルーピング・集計し、SVGのドーナツグラフとして可動描画。
    *   予算カテゴリに設定されたグラデーションクラス名（例: `from-orange-500 to-amber-400`）をカラーコードマップ（`colorMap`）を介してSVGグラデーションタグに変換し、滑らかなグラデーション付きのグラフを表示。

### 💰 3.3. 予算設定 (Budgets)
*   **予算カテゴリ管理**:
    *   ユーザーごとに複数の予算カテゴリ（例: 食費、家賃、交際費など）を作成可能。
    *   カテゴリ名、月間予算（金額）、およびカラーパレット（Tailwindのグラデーションクラス）を設定可能。
*   **CRUD操作**:
    *   予算の新規作成、既存予算の編集、予算の削除機能を実装。`src/lib/supabase/budgets.ts` がバックエンドへのデータ同期を担当。

### 💸 3.4. 支出管理 (Expenses)
*   **出費の記録と一覧**:
    *   金額、日付、カテゴリ、メモ（任意）を入力して、日々の出費を記録するフォームをモーダルで実装。
*   **当月の出費一覧**:
    *   表示中の年月に合致する支出データをリスト表示。
*   **来月以降の予想出費一覧**:
    *   今日（サーバー日時）より未来の日付で登録された予定支出（定期の引き落とし予約など）を検知し、「予想出費」として別枠で一覧化。
*   **予算消化状況（プログレスバー）**:
    *   各予算カテゴリに紐づく当月の支出合計を計算（`reduce` メソッドによる集計処理）。
    *   各カテゴリの進捗バーを表示し、予算額を超過（オーバー）した場合は使用金額が赤文字に変化。

## 4. データアクセスと状態管理の仕組み

*   **Supabase DAL**:
    *   クライアントコンポーネントからは `src/lib/supabase/client.ts` の `createBrowserClient`、サーバーコンポーネントや Server Actions からは `src/lib/supabase/server.ts` の `createServerClient` を適切に使い分けてセッションおよびデータを取得。
*   **非同期データのロードと並行処理**:
    *   ダッシュボードや支出管理画面では、読み込み速度の向上のため、`Promise.all` を使用して予算と支出のデータを並行でフェッチしています。
    *   未ロード時のガタツキを防ぐため、スケルトン表示（読み込み中プレースホルダー）を搭載。
